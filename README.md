## 🧠 Overview
This project presents an interactive **Amazon Sales Dashboard** built using Power BI to analyze sales performance, customer behavior, and business trends.

The goal of this project is to transform raw sales data into meaningful insights that help in **data-driven decision-making**.

---

## 🎯 Objectives
- Analyze sales trends over time
- Identify top-performing products and categories
- Understand customer purchasing patterns
- Track KPIs like revenue, profit, and order volume
- Build a visually appealing and interactive dashboard

---

## 🛠️ Tech Stack
- **Power BI** – Data visualization & dashboard creation  
- **Excel / CSV** – Data source  
- **Power Query** – Data cleaning & transformation  
- **DAX (Data Analysis Expressions)** – Calculations & KPIs  

---

## 📈 Key Features
- 📊 Interactive dashboard with filters & slicers  
- 📅 Time-based sales analysis (daily, monthly, yearly)  
- 🏆 Top products & category performance insights  
- 💰 Revenue & profit tracking  
- 🌍 Region-wise / segment-wise analysis  
- 📉 Trend analysis using charts and graphs
